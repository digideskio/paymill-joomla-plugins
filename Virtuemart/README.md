Virtuemart Installation Instructions
-------------

- Install component com_paymillapi
- Install plugin plg_paymill
- From the Joomla Administrator, specify Public & Private keys in Paymill plugin configuration
- Go to Go to Components->VirtueMart->Shop->Payment Methods : Create new paymenthod for paymill with Name- 'Paymill' & Select Paymethod 'VM Payment Paymill' from dropdownlist.
- Publish this new Payment Method
